# Rule of Third

CEP-плагин-панель для Adobe After Effects, предназначенный для построения композиционных сеток на рабочей области и автоматического размещения медиафайлов по зонам сетки с центрированием и масштабированием.

Разработан для студии **ООО «ГраВерД»**.

---

## Возможности

- **Сетка «правило третей»** (3×3, равномерное деление)
- **Сетка «золотое сечение»** (3×3, линии расположены на 38.2% и 61.8%)
- **Кастомная сетка** с произвольным числом строк и колонок (до 8×8)
- **Гибкие веса зон** — каждая колонка и строка настраивается ползунком в диапазоне 1–5
- **Отображение сетки в композиции AE** в виде Shape Layer, а также её скрытие
- **Размещение медиафайла** из Project Panel в одну или несколько ячеек сетки
- **Выделение прямоугольного диапазона ячеек** перетаскиванием (drag)
- **Автоматическая синхронизация ориентации** панели с активной композицией
- **Сохранение пользовательских сеток** в localStorage с привязкой к ориентации, а также их загрузка и удаление
- **Очистка назначений ячеек**

## Установка

Плагин распространяется в виде установочного пакета `RuleOfThird.zxp` и устанавливается через **ZXP Installer**.

1. Скачайте файл `RuleOfThird.zxp` из репозитория проекта.
2. Установите [ZXP Installer](https://aescripts.com/learn/zxp-installer/) (или аналогичный установщик расширений Adobe), если он ещё не установлен.
3. Откройте ZXP Installer и перетащите в его окно файл `RuleOfThird.zxp` либо выберите его через диалог открытия файла.
4. Дождитесь завершения установки — расширение будет автоматически скопировано в системную папку расширений CEP.
5. Перезапустите Adobe After Effects.
6. Откройте панель через меню **Window → Extensions → Rule of Third**.

Установка через ZXP Installer не требует включения режима отладки (Debug Mode) и подходит для конечных пользователей.

## Использование

1. Откройте панель **Rule of Third** в After Effects (Window → Extensions → Rule of Third).
2. Панель автоматически определяет ориентацию активной композиции и подбирает соответствующую сетку.
3. Выберите тип сетки: правило третей, золотое сечение или кастомная.
4. Для кастомной сетки задайте количество строк и колонок и настройте веса зон ползунками.
5. Нажмите «Показать сетку», чтобы отобразить сетку в окне композиции AE как Shape Layer. Кнопка «Скрыть сетку» убирает её из композиции.
6. Чтобы разместить файл в зоне сетки:
   - выделите нужный файл в Project Panel After Effects;
   - выделите одну ячейку или диапазон ячеек на canvas панели (перетаскиванием);
   - нажмите «↓ Разместить файл» — файл будет помещён в композицию с автоматическим центрированием и масштабированием по выбранным зонам.
7. Кнопка «✕ Очистить ячейки» сбрасывает назначения зон без удаления слоёв из композиции.
8. Настроенную кастомную сетку можно сохранить с указанием названия — она появится в выпадающем списке сохранённых сеток с пометкой ориентации (альбомная/портретная) и доступна для повторной загрузки или удаления.

## Структура проекта

```
RuleOfThird/
├── CSXS/
│   └── manifest.xml        — паспорт плагина для Adobe CEP
├── css/
│   └── style.css           — стили панели
├── js/
│   ├── CSInterface.js      — библиотека Adobe для связи UI и AE
│   └── main.js             — логика панели (UI, canvas, события)
├── jsx/
│   └── hostscript.jsx      — ExtendScript-код, выполняется внутри AE
└── index.html              — разметка панели
```

---

## Архитектура

Плагин реализован по трёхуровневой архитектуре:

- **UI Layer** — HTML/CSS/JavaScript, отрисовка сеток и взаимодействие с пользователем через Canvas API.
- **Bridge Layer** — `CSInterface.js`, обеспечивает связь между панелью (UI Layer) и средой After Effects.
- **AE Layer** — `hostscript.jsx`, ExtendScript-код, выполняющий построение сетки в композиции и размещение медиафайлов.

---

## Зависимости

Библиотека `CSInterface.js` поставляется в составе плагина в каталоге `js/CSInterface.js` и используется без модификаций.
